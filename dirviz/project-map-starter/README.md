# Project Map Starter

A GitHub Pages-ready starter app that lets a user upload a ZIP file and view the project folder/file structure as an interactive node graph.

## Files

- `index.html`
- `style.css`
- `app.js`

## How to use

1. Open `index.html` in a browser.
2. Upload a `.zip` file.
3. Explore the generated graph.
4. Click a node to inspect it.
5. Double-click a folder node to collapse or expand it.

## Libraries

This starter uses CDN links for:

- JSZip
- D3.js

## Classroom ideas

- Have students upload a ZIP of their project.
- Screenshot the structure.
- Explain what each major folder does.
- Improve the project organization and compare before/after maps.
